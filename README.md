# No-Piano
영상처리를 활용한 피아노 장갑 개발
<br/><br/>

## 프로젝트 설명

피아노가 없이 카메라와 장갑을 이용해서 피아노를 연주하는 것을 목표로 합니다.<br/>
장갑에 터치센서와 자이로센서를 부착하고 각 손가락에 색을 입힙니다.<br/>
라즈베리파이와 어플리케이션을 블루투스로 연결합니다. <br/>
터치센서가 동작할 때 해당 손가락의 색을 영상처리(OpenCV)를 사용해서 위치를 찾습니다.<br/>
해당 위치에 해당하는 건반을 어플리케이션을 통해서 시각화하고 소리를 출력합니다.<br/>
자이로 센서를 활용해서 악기를 바꾸거나, 옥타브를 바꾸는 기능도 가능합니다.<br/><br/>
시연 영상 : https://youtu.be/4k9JiGBaHqQ <br/><br/>
